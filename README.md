# 💳 흥청망청 가계부 🐷

가계부를 단순 기록으로 끝내지 않고,  
**AI 분석 + 물가 평균가 비교 + 저축 챌린지**로 이어지게 만든 금융 라이프 웹 서비스입니다.

---

## 📆 프로젝트 기간
- 2025.11.14 ~ 2025.12.25

---

## 📌 프로젝트 개요
본 프로젝트는 사용자가 수입/지출 내역을 기록한 뒤,

- 월별 소비 패턴을 시각적으로 확인하고,
- AI 코멘트로 소비 습관을 점검하며,
- 내가 지불한 가격이 평균 대비 합리적인지 판단하고,
- 챌린지로 저축 습관까지 연결할 수 있도록

설계된 **풀스택 금융 서비스**입니다.

---

## ⭐ 프로젝트 핵심 기능

### 1) 🔐 인증/사용자
- 회원가입, 로그인(JWT)
- 마이페이지 조회/수정
- 비밀번호 변경

### 2) 📒 가계부(Book)
- 수입/지출 CRUD
- 월별 필터 조회
- 카테고리별 합계 통계
- AI 재무 분석 리포트 생성

### 3) 📊 월별 통계 대시보드
- 월 이동(이전/다음)
- 총수입 / 총지출 / 잔액 요약
- 카테고리 도넛 차트
- AI 분석 결과 표시

### 4) 🛒 평균계산기(물가 비교)
- 상품 키워드 + 업태 + 사용자 지출 입력
- DB 평균가와 비교해 소비 판정
- 데이터 부족 시 AI 추정 가격 + 팁 제공

### 5) 🎯 챌린지/저축
- 챌린지 생성/조회/수정/삭제
- 챌린지별 입금 등록/조회/수정
- 누적 입금액 및 횟수 요약

### 6) 💬 커뮤니티
- 게시글 작성/조회/수정/삭제
- 좋아요 토글, 조회수 증가
- 댓글 CRUD

## 🛠️ 주요 기술 스택

### Frontend
- Vue 3
- Vite
  
### Backend
- Java 17
- Spring Boot 3
- Spring Security + JWT
- MyBatis
- MySQL

### AI 연동
- OpenAI Gateway 연동(프롬프트 기반 응답 생성)

```text
Vue3 SPA
 ├─ Router Guard
 ├─ Pinia Store
 └─ Axios API Call
      ↓
Spring Boot API
 ├─ SecurityFilterChain + JWT Filter
 ├─ Controller
 ├─ Service
 └─ MyBatis Repository
      ↓
MySQL
```
